## 🎯 Quick Start

### For Beginners (Shared Hosting)

1. **Read:** DEPLOYMENT_SUMMARY.md
2. **Follow:** DEPLOYMENT_CHECKLIST.md → Section: "Option B: Manual Deployment"
3. **Upload** files via FTP
4. **Configure** .env file
5. **Run** setup commands via SSH

### For Developers (VPS/Cloud)

1. **Clone** repository to server
2. **Copy** env.production.template to .env
3. **Run** ./deploy.sh
4. **Configure** web server
5. **Install** SSL certificate

### For Quick Deploy (PaaS)

1. **Connect** to Laravel Forge/Ploi
2. **Deploy** via Git
3. **Done!**

Before going live, ensure:

```env

APP_DEBUG=false            # ✅ Must be false
APP_KEY=base64:...         # ✅ Generate new key
APP_URL=https://...        # ✅ Use HTTPS
```

- [ ] Strong database password
